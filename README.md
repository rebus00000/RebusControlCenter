# Rebus Control Center

A modern, customizable Android Control Center built with **Kotlin** and **Jetpack Compose**.

Rebus Control Center is designed to provide a fast, elegant, and highly customizable quick settings experience. It focuses on smooth animations, premium UI design, and efficient performance while remaining lightweight and battery-friendly.

---

## ✨ Features

### Core Features
- Edge swipe gesture
- Floating Control Center overlay
- Brightness control
- Volume control
- Music player controls
- Quick settings tiles
- Battery information
- Network information
- Quick app shortcuts

### Customization
- Custom wallpapers
- Theme engine
- AMOLED mode
- Light & Dark themes
- Material You support
- Custom icon packs
- Adjustable transparency
- Glassmorphism effects
- Accent color customization

### Smart Features
- Dynamic media artwork
- Notification support
- Widget support
- RAM usage
- Battery statistics
- Custom layouts
- Drag & drop tile editing
- Multi-page Control Center

### Performance
- Lightweight architecture
- Optimized animations
- Low battery usage
- Fast startup
- Minimal background activity
